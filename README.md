# Privacy Shield

Privacy Shield is a full-stack privacy monitoring platform designed to help users understand how online trackers collect and use their data.

Many websites contain hidden trackers that monitor your browsing behavior, shopping activity, search history, and interests. This information is often used to build advertising profiles and can be shared with advertising networks and third-party platforms. As a result, users frequently see advertisements for products they recently searched for or viewed across different platforms such as YouTube, Instagram, Facebook, and other websites.

Privacy Shield helps users regain visibility into this process by detecting trackers, analyzing privacy risks, monitoring tracking activity, and blocking selected tracking domains through a Chrome Extension.

## Key Features

* Detect hidden website trackers in real time
* Monitor privacy risks and advertising exposure
* Analyze tracker activity through dashboards and analytics
* Scan domains for suspicious tracking behavior
* Receive notifications about detected trackers
* View tracker history and recent activity
* Block selected tracking domains using a Chrome Extension
* Clear detected tracker records using Vanish Mode

## Project Goal

The goal of Privacy Shield is to educate users about online tracking, increase transparency around data collection practices, and provide tools to help users protect their privacy while browsing the web.
